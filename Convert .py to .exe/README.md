# Convert .py to .exe (Tkinter Digital Clock) 🕰️💻

This mini-project demonstrates how to convert a Python script (a Tkinter-based Digital Clock application) into a standalone Windows executable (`.exe`) file. This allows the application to run on machines without requiring a Python interpreter or any dependencies installed.

## 🚀 Features
- Includes a fully functional Digital Clock built using Tkinter.
- Displays Time (Hours, Minutes, Seconds, AM/PM) and Date (Day, Month, Year).
- Uses a background image for an enhanced GUI.
- Contains a `main.spec` file generated by PyInstaller, illustrating the packaging configuration.

## 🛠️ Prerequisites
To convert Python scripts to executables, you need the `pyinstaller` library.

1. **Install PyInstaller:**
   ```bash
   pip install pyinstaller
   ```
2. **Additional Libraries** (for the clock itself):
   ```bash
   pip install pillow
   ```

## 💻 How to Run (and Convert)
To convert the `main.py` script into an executable, follow these steps:

1. Open a terminal in the project directory.
2. Run the PyInstaller command:
   ```bash
   pyinstaller --onefile --windowed main.py
   ```
   *Note: `--onefile` packages everything into a single executable, and `--windowed` hides the console window when running GUI applications.*
3. After the build finishes, you will find your executable inside the `dist` folder.
4. If you want to run the clock directly as a Python script, use:
   ```bash
   python main.py
   ```

## 📂 Project Structure
- `main.py`: The Tkinter digital clock source code.
- `main.spec`: The PyInstaller configuration file used for the build process.
- `background.jpg`: The background image used for the Tkinter UI.
