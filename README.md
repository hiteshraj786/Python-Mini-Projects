# Python Mini Projects 🐍🔥

Welcome to the **Python Mini Projects** repository! This collection contains 45 diverse and practical mini-projects built using Python. These projects cover various domains such as GUI development, automation, video processing, PDF manipulation, web scraping, and more.

## 🚀 Projects Included

1. **Add Watermark in Video Using Python** - Add text overlays to video clips.
2. **Automate WhatsApp Message Using Python** - Schedule and send WhatsApp messages automatically.
3. **Convert .py to .exe** - Learn how to compile Python scripts into standalone executables.
4. **Convert Pdf to Image Using Python** - Convert each page of a PDF into JPEG format.
5. **Convert Pdf to Word and Word to Pdf Using Python** - Bi-directional conversion between PDF and MS Word formats.
6. **Create Split Screen Video using Python** - Merge multiple video feeds into a split-screen layout.
7. **Desktop Notifier** - A local notification system for health/hydration reminders.
8. **Desktop Notifier to Display Stock Market Data Using Python** - Real-time stock alerts on your desktop.
9. **Dice Roll Simulator Using Python** - A GUI based virtual dice roller.
10. **Digital_Clock** - A visually appealing digital clock built with Tkinter.

*...and 35 more exciting projects covering:*
- **File Management & Conversion** (e.g., File Manager, MP4 to MP3, Rotate PDF).
- **Automation** (e.g., Skype Automation, InstaBot).
- **Utilities** (e.g., Internet Speed Test, Take Screenshot, Website Blocker).
- **Games & AI** (e.g., Snake Game, Hangman, Jarvis Voice Assistant, Face Recognition).

## 🛠️ Getting Started

Each mini-project has its own folder containing the source code and a specific `README.md`. 
1. Navigate to the project folder you are interested in.
2. Check the `README.md` inside that folder for specific prerequisite libraries (e.g., `pip install <library>`).
3. Run the script!
